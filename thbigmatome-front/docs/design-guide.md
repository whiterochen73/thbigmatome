# thbigmatome デザインガイド

**バージョン**: 1.0
**作成日**: 2026-02-21
**対象**: フロントエンド（Vue.js 3 + Vuetify 3）

---

## 概要

**テーマコンセプト**: 東方 × 野球融合 — 和風の落ち着き × スポーツの躍動感
**ターゲット**: PC優先、CUIゲーマー・ベテラン野球ファン向け
**情報設計方針**: 高密度テーブル主体、視認性と操作効率を両立する

## 2. タイポグラフィ

### フォント選定

```
見出し系 : Shippori Mincho (Google Fonts) — 明朝体、和風の格調
本文・UI : Noto Sans JP — 可読性高く、UI操作に最適
数値・統計: JetBrains Mono / Noto Sans JP — 等幅で統計表が読みやすい
```

### フォントサイズ体系

| 用途 | クラス | サイズ | 行間 | 用例 |
|------|--------|--------|------|------|
| 大見出し | `text-h4` | 2.125rem | 1.3 | ページタイトル |
| 中見出し | `text-h5` | 1.5rem | 1.35 | セクション名 |
| 小見出し | `text-h6` | 1.25rem | 1.4 | カード見出し |
| 本文 | `text-body-2` | 0.875rem | 1.5 | 説明文・通常テキスト |
| 統計数値 | `text-body-2` (mono) | 0.875rem | 1.4 | 打率・防御率等 |
| キャプション | `text-caption` | 0.75rem | 1.4 | 補足・ラベル |
| ラベル | `text-overline` | 0.625rem | 2.0 | テーブルヘッダー |

### 数値表示ルール

- 打率・防御率: `.000` 形式（3桁小数点）
- 本塁打・打点等: 右揃え整数
- 投球数・イニング: 右揃え
- すべての統計数値は等幅フォント推奨

## 3. レイアウト方針

### グリッドシステム

```
全体幅: 最大 1440px（PC最適）
サイドナビ (展開時): 256px
サイドナビ (縮小時): 72px（rail mode）
メインコンテンツ: 残余幅
モバイル対応: ≥ 768px でナビ表示（モバイルは縮小モード）
```

### ナビゲーション構造

```
┌──────────────────────────────────────────┐
│  AppBar: [☰] 東方BIG野球まとめ    [⚙][👤] │
├──────────┬───────────────────────────────┤
│  SideNav │  MainContent                  │
│  (256px) │                               │
│  [🏠]ホーム         │  <router-view>      │
│  [👥]チーム管理     │                     │
│  [👤]選手一覧       │                     │
│  [⚾]試合記録       │                     │
│  [📊]成績まとめ     │                     │
│  [💰]コスト管理     │                     │
│  ──────── │                              │
│  [🏆]リーグ管理(C)  │                     │
│  ──────── │                              │
│  [◀]縮小           │                     │
└──────────┴───────────────────────────────┘
```

### 情報密度ルール

- **テーブル**: `density="compact"` を基本とする（行高 40px）
- **カード**: `pa-3`〜`pa-4`（過度なパディングを避ける）
- **ダイアログ**: 最大幅 800px（モバイルは全幅）
- **ページ余白**: `<v-container>` の標準パディングを使用

### Zインデックス体系

```
AppBar      : z-index 100
SideNav     : z-index 90
ダイアログ  : z-index 200
スナックバー: z-index 300
ツールチップ: z-index 250
```

### コンポーネントカスタマイズ方針

| コンポーネント | 設定方針 |
|---------------|---------|
| `v-data-table` | `density="compact"`, ヘッダー背景色: primary lighter |
| `v-app-bar` | `color="primary"`, `elevation="2"`, 高さ64px |
| `v-navigation-drawer` | rail対応, `rail-width="72"` |
| `v-btn` | Primary action: `color="accent"` variant="flat" |
| `v-chip` | 統計ラベル用途: `size="small"` density="compact" |
| `v-card` | `elevation="1"` または `variant="outlined"` |
| `v-snackbar` | `location="top"` timeout=3000 |
| `v-tooltip` | `location="bottom"` delay=200ms |

### グローバルデフォルト（`defaults` オプション）

```ts

  VDataTable: { density: 'compact', hover: true },
  VBtn: { variant: 'tonal' },
  VCard: { elevation: 1 },
  VTextField: { density: 'compact', variant: 'outlined' },
  VSelect: { density: 'compact', variant: 'outlined' },
}
```
