# 選手カードの仕組み

東方BIG野球の選手はすべて「カード」として管理されています。カードには能力値が設定されており、試合中の判定に使われます。

## カード種別

選手カードは **投手カード** と **野手カード** の2種類です。投手と野手では試合中の役割が根本的に異なるため、別々のカードとして管理されます。

同一キャラクターが投手カードと野手カードの両方を持つこともあります（いわゆる二刀流）。

## 能力値

### 投手の能力値

#### スタミナ

投手が投げられるイニング数に影響する能力値です。

- **先発投手**: 4〜9（長いイニングを投げるため高めに設定）
- **中継ぎ投手**: 0〜3（短いイニング専門のため低め）

先発投手が5イニング未満で交代した場合は「KO（ノックアウト）」扱いとなり、勝利投手の権利を得られません。

### 野手の能力値

#### 走力（1〜5）

走塁能力を5段階で表します。盗塁判定やエンドラン（ヒットエンドラン）の成否に影響します。

#### バント（1〜10）

バント能力を10段階で表します。値が高いほどバントが成功しやすくなります。

#### 盗塁開始値（1〜22）

ダイス判定との比較により、盗塁を試みるかどうかを決定する値です。

#### 盗塁完了値（1〜22）

盗塁開始値と組み合わせて、盗塁が成功するかどうかの判定に使われます。

### 共通の能力値

#### 負傷率（0〜7）

選手が負傷する確率を表します。0は負傷しないことを意味します。UP表（好プレイ珍プレイ表）の判定で使用されます。

## バリエーション

通常、同一カードセット内で同じキャラクターの同種カード（投手なら投手）は1枚だけです。

しかし、一部のカードセットでは同一キャラクターの同種カードが複数存在します。たとえば「PM2026」カードセットには、同じ投手キャラクターの「湘南バージョン」「町田バージョン」「URバージョン」といったバリエーションがあります。

バリエーションはラベル（文字列）で区別されます。バリエーションが設定されていない通常カードは、従来通り1キャラクター1枚の制限が適用されます。

## カードセット

選手カードは「カードセット」という単位で管理されています。異なるカードセット間では、同一キャラクターの複数カードを持つことができます。カードセットが異なれば、同じキャラクターでも能力値が異なる場合があります。

## 打順

試合では9人打線が基本です。DH（指名打者）制を採用する場合でも、打順の枠は1番から9番の9枠です。
