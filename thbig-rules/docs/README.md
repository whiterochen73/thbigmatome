# 東方BIG野球 ルールガイド

東方BIG野球は、IRCチャット上で行われるテキストベースの野球シミュレーションゲームです。プレイヤーは東方Project・ハチナイ・タマヨミなどの作品からキャラクターを集めてチームを編成し、IRC上で自動進行する試合を楽しみます。

## ゲームの2つのレイヤー

東方BIG野球のルールは、大きく2つのレイヤーに分かれています。

### ゲーム本体（東方BIG野球）

IRCチャットで行われる野球ゲームそのもののルールです。打席処理の仕組み、投手のスタミナ、盗塁判定など、1試合の中で起こることはすべてこのレイヤーに含まれます。

### 運営ルール（Lペナ）

「IRCリーグ 東方BIG野球ペナントレース」の運営ルールです。チームの構成制限（コスト上限・人数制限・外の世界枠）、シーズン管理、大会ルール、監督制度など、試合の外側にあるリーグ全体の仕組みを定めています。

## 機械可読データ

AIエージェントや実装者が参照するための構造化ルールデータは `game_rules.yaml` にあります。各ルールには理由（`why`）・例外（`exception`）・関連ルール（`related`）が付記されており、ルールの背景まで含めて正確に参照できます。

このドキュメント群は `game_rules.yaml` の内容を人間向けに解説したものです。ルールの正確な値は `game_rules.yaml` を参照してください。
