# sources/ — Wiki以外のデータソース・調査メモ

このディレクトリには、Wikiスクレイプ以外のルール出典・調査メモ・口頭確認ルール等を置く。

## 収録方針

- **P口頭確認ルール**: コミッショナーから口頭で確認されたルールの記録
- **調査メモ**: Wiki等を調査した際の分析・解釈メモ
- **仕様書素材**: ルールブック執筆の前段階にあたる素材文書
- **外部ソース引用**: ゲーム外の参考資料（IRCログ分析等）
