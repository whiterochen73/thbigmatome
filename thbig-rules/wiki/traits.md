# 特徴・特殊能力一覧
出典: https://thbigbaseball.wiki.fc2.com/wiki/%E7%89%B9%E5%BE%B4

## ⚠️ 重要な注記

a) **このページに掲載されているのはよくある（所有者が多い）特徴・能力のみ。全てではない。**
b) **ニナのように複雑な固有特徴を持つ選手も存在する。**
c) **カード上は「特徴」として記載されていても、実装上は「特殊能力」として扱うものがある。**

---

## 投手特徴

投手特徴は「UP表で投手特徴確認を引くと発動」します。
野手が登板時（投手特徴がカードに記載されていない場合）は「ホームラン」になります。

| 特徴 | 効果 |
|------|------|
| WP | 暴投で、各ランナーが1つずつ進塁してもう一度投球。走者がいないときは投球から振り直し |
| DB | 死球。この死球では怪我や危険球退場（警告試合を除く）は発生しない |
| ボーク | ボークで、各ランナーが1つずつ進塁してもう一度投球。走者がいない時は投球から振り直し |
| ホームラン | ホームランを被弾 |
| けが | 投手は負傷して、自分の怪我レベルで怪我チェックを行う |
| けが＊＊日 | 投手はその日数分怪我をして降板。投手交代が必要 |
| 対右 | 打者は右打者なら5、左打者なら1で打つ |
| 対左 | 打者は右打者なら1、左打者なら5で打つ |

---

## 投手特殊能力

| 能力 | 効果 |
|------|------|
| WPしない | あらゆる暴投を防ぐ。UP表での暴投は投球から振り直し。パスボールは防げない |
| ボークしない | ボークを防ぐ。UP表でのボークは投球から振り直し |
| 精神疲労 | 通常の疲労に加え、失点するとその時点で疲労状態になる |
| クイック○ | 盗塁されにくくなる。盗塁のst値出目が+2、捕手肩が+2 |
| クイック✕ | 盗塁されやすくなる。盗塁のst値出目が-2、捕手肩が-2 |

---

## 野手特徴

野手特徴は「UP表で打者特徴確認を引くと発動」します。
投手の打席（野手特徴がカードに記載されていない場合）では「三振」になります。

| 特徴 | 効果 |
|------|------|
| DB | 死球。怪我や危険球退場は発生しない |
| DBけが＊＊日 | 死球で日数分怪我。代走が必要 |
| DB乱闘 | 死球で乱闘となり打者退場。投手は怪我レベル6で怪我チェック |
| けが＊＊日 | 打者は日数分負傷。代打が必要 |
| 1C5 | ランナーなし時は1、ランナーあり時は5で振り直し |
| チャンス | ランナーなし時は5、ランナーあり時は1で振り直し |
| 内野安打 | 内野安打。1塁走者がいれば2塁フォースアウト。内野前進守備時は3塁走者がホームでアウト |
| 対右 | 相手が右投手なら1、左投手なら5で振り直し |
| 対左 | 相手が右投手なら5、左投手なら1で振り直し |
| （代打） | 代打時は1で振り直し。代打でない時は他の特徴を使用 |

## 野手特殊能力

| 能力 | 効果 |
|------|------|
| 風神少女 | 走塁表の出目を-2。出目20は補正を受けずアウト |

## IRC実装メモ
- 特徴の発動タイミング: UP表で「投手/打者特徴確認」を引いた時
- 野手が投手として登板時: 投手特徴は「ホームラン」に変換
- 投手が打席に立つ時: 野手特徴は「三振」に変換
- `けが＊＊日`: `**`の部分は具体的な日数（カード記載の数字）に置換
- 「対右」「対左」特徴: 投打双方に存在し、効果が異なる
- クイック○/✕: 盗塁判定時のst値と捕手肩への補正値（±2）
- 風神少女: 走塁表出目を-2補正するが20はアウト確定
