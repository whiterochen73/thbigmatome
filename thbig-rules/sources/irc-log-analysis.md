## sianの二重性

`sian` はボットを起動しているプレイヤーでもある。

- **ボット応答**: `(sian) (triggering_user) ...` の形式（括弧内にトリガーユーザー名がある）
  - 例: `(sian) (lemon) 野崎 Dice => [14] result:５／４`
  - 例: `(sian) (lemon) 球場【札幌ドーム...】`
- **プレイヤー発言**: `(sian) テキスト` の形式（括弧なし、直接テキスト）
  - 例: `(sian) 三ゴ　試合終了`
  - 例: `(sian) ありがとうございました。`
- **判別方法**: `(sian) (` で始まればボット応答、それ以外はプレイヤー発言
- 注意: `sian0` というユーザー名の場合もある（`(sian0) (Aal) マミゾウ Dice => ...`）

## 試合境界パターン

### 開始マーカー

プレイヤーが `試合ルール` と発言 → sianボットが3行で応答:

```
HH:MM:SS (player) 試合ルール
HH:MM:SS (sian) (player) 球場【球場名 1112：○○ 1314：○○】
HH:MM:SS (sian) (player) DH【有り/無し】 設定【M月D日】季節【○○：キャラ名】季節【○○：キャラ名(効果)】
HH:MM:SS (sian) (player) 試合開始時間【HH:MM】
```

抽出可能情報:
- 球場名、UP表参照(1112/1314)
- DH有無
- ゲーム内設定日付
- 季節（二十四節気）とバイオリズム対象キャラ
- 試合開始時間

### 終了マーカー

最後のプレイ結果に `試合終了` を含む:

```
HH:MM:SS (player) {プレイ結果}　試合終了{スコア等}
HH:MM:SS (player_A) ありがとうございました{。等}
HH:MM:SS (player_B) ありがとうございました{。等}
```

バリエーション:
- スコア形式: `1-6`, `３－４`, `7x-6`(サヨナラx付き), スコアなし
- 句読点: `試合終了！`, `試合終了　`, `試合終了`
- 発言者: プレイヤーまたはsian（プレイヤーとして）

### エッジケース

1. **日をまたぐ試合**: ログファイルは日次分割。試合が0時をまたぐと別ファイルに分断。
   - 例: 9/1 22:46開始 → 9/2 00:00に分断、9/2冒頭に`ありがとう`
   - 対応: 隣接ファイルの連結処理が必要
2. **試合終了の明示なし**: 稀に`試合終了`と書かずに`併殺。`等で暗黙終了するケース
   - 対応: `試合ルール`開始後に次の`試合ルール`が来たら前の試合は終了とみなす
3. **不完全な試合**: IRCクライアント未起動で開始/終了が欠落
   - 対応: 開始マーカーのない試合はスキップ
4. **1ファイル複数試合**: 大会日などは1日5試合以上ある
5. **怪我チェック**: `1d20 怪我チェック` が試合終了後〜次試合開始前に出現

## 打席進行パターン

基本フロー（1打席）:
```
(pitcher_player) XX-p          ← 投球指示（マクロコマンド）
(sian) (pitcher_player) 投手名 Dice => [N] result:投球番号  ← 投球結果
(batter_player) YY-N           ← 打撃指示（投球番号を指定）
(sian) (batter_player) 打者名 Dice => [N] result:打撃結果   ← 打撃結果
(player) プレイ結果テキスト     ← 人間によるプレイ結果記述
```

- 投球コマンド: `XX-p` (XXはマクロ番号、pは投球を意味)
- 打撃コマンド: `YY-N` (YYはマクロ番号、Nは投球番号)
- sianのDice応答: `キャラ名 Dice => [出目] result:結果`

### ダイスコマンド

- `1d20`: 汎用ダイス（先攻後攻、怪我チェック等）
- `1D20`: 同上（大文字小文字混在あり）

### イニング区切り

```
(player) {プレイ結果}　{N}回{表/裏}終了{スコア}
```

表記揺れ:
- 漢数字: `一回表終了`, `九回裏終了`
- アラビア数字: `1回表終了`, `9回裏終了`
- スコア形式: `0-0`, `3-`, `0-4`（片方省略あり）
