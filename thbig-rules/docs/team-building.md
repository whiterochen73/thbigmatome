# チーム構成ルール

東方BIG野球でチームを作るにあたって知っておくべきルールをまとめています。

## ロスター（登録選手枠）

チームに登録できる選手の人数は **25人以上50人以下** です。

- **下限25人**: 1軍の試合を成立させるために必要な最小人数です
- **上限50人**: 管理可能な規模の上限として設けられています

なお、シーズン前の一時保存段階（まだ1試合も行っていない状態）では、下限チェックはスキップされます。

## コスト制度

すべての選手カードにはコストが設定されており、チーム全体のコスト合計には上限があります。強力な選手ばかりを集められないようにすることで、チーム間のバランスを保っています。

### チーム合計コスト

チームに所属する全選手のコスト合計は **200以下** でなければなりません。

ただし、助っ人枠など一部の特殊な選手はコスト計算の対象外です。

### 1軍コスト（スライド制）

1軍に登録する選手のコストには、登録人数に応じた上限があります。1軍の規模が大きいほど、使えるコストの上限も上がる仕組み（スライド制）です。

| 1軍人数 | コスト上限 |
|---------|-----------|
| 28人以上 | 120 |
| 27人 | 119 |
| 26人 | 117 |
| 25人 | 114 |

1軍は最低25人必要です。25人未満では登録（シーズン開始）ができません。

## 外の世界枠

チームの主要シリーズ以外のキャラクター（「外の世界」の選手）は、1軍に **最大4人** まで登録できます。これはゲームの世界観を維持するためのルールです。

4人フル活用する場合は、投手1名以上・野手1名以上を含む混成にする必要があります。これにより、外の世界の選手に偏った戦術を防ぎ、多様な編成を促しています。

どのキャラクターが「外の世界」に該当するかは、チーム種別によって異なります（後述）。

## チーム種別

チーム作成時に種別を選択します。種別は後から変更できません。

### 通常チーム（normal）

東方Projectのキャラクターが「内の世界」（ネイティブ）です。それ以外のシリーズのキャラクターが外の世界枠にカウントされます。

### ハチナイチーム（hachinai）

ハチナイ・タマヨミのキャラクターが「内の世界」です。東方Projectのキャラクターが外の世界枠にカウントされます。通常チームとは外の世界の定義が逆転する形です。

### オリジナル選手

特定のシリーズに属さないオリジナル選手は、チーム種別に関わらず常に外の世界枠として扱われます。

## セカンドチーム

1人のオーナーは **最大2チーム** を持つことができます。

ただし、同一選手を両方のチームに登録することはできません（選手排他制約）。同じオーナーのチーム間でも、選手は1チームにしか所属できません。
