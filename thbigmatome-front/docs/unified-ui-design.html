実装ガイド

1. ページ構造の統一ルール

全ページが以下の4層構造を守ること。層の省略は可（例: フィルター不要ならFilterBar省略）、層の順序変更・独自レイアウトは不可。

  • Layer 1 — PageHeader: 既存 PageHeader.vue を使用。タイトル左に朱色3pxバー。右端にアクションボタン
  • Layer 2 — FilterBar: 既存 FilterBar.vue を使用。PlayerCardsViewのインラインフィルターを廃止してこちらに統一
  • Layer 3 — DataCard: 既存 DataCard.vue 内に v-data-table。カスタムリスト形式は原則廃止しテーブルに統一
  • Layer 4 — Pagination: v-data-table 標準ページネーション。右寄せ、Items per page付き
<!-- 全ページの基本テンプレート --> <template> <v-container> <PageHeader :title="t('page.title')"> <template #title-actions> <v-btn color="accent">...</v-btn> </template> </PageHeader> <!-- タブがある場合のみ --> <v-tabs v-model="tab" color="primary">...</v-tabs> <FilterBar> <template #search> <v-text-field ... /> </template> <template #filters> <v-select ... /> </template> <template #toggles> <span class="result-count">{{ count }}件</span> </template> </FilterBar> <DataCard> <v-data-table :headers :items density="compact" hover /> </DataCard> </v-container> </template>

2. PageHeader 改修ポイント

既存の PageHeader.vue に朱色左バーを追加。h1のフォントを Shippori Mincho に明示。

  • タイトル左に border-left: 3px solid var(--shu) + padding-left: 16px
  • タイトルフォント: font-family: var(--font-display), font-size: 22px
  • マージン下: margin-bottom: 20px
  • アクションボタン: 右端に flex 配置（既存通り）

3. FilterBar 改修ポイント

既存の FilterBar.vue にカード風背景とボーダーを追加。slot構成は維持。

  • 背景: var(--bg-elevated) + border: 1px solid var(--border-light) + border-radius: 8px
  • パディング: 12px 16px
  • 影: box-shadow: var(--shadow-card)
  • 結果件数は #toggles スロットに統一配置（JetBrains Mono, 12px, caption色）
  • PlayerCardsView.vue のインラインフィルターを廃止し、FilterBarの各slotに移行

4. DataTable スタイル統一

Vuetifyの v-data-table にグローバルCSSオーバーライドを適用。

  • ヘッダー背景: var(--bg-inset) (#F0EDE5)
  • ヘッダーテキスト: 12px, font-weight: 600, color: var(--text-medium)
  • 行ストライプ: 偶数行に var(--bg-stripe)
  • ホバー: rgba(27, 58, 107, 0.04)
  • 数値列: text-align: right + JetBrains Mono
  • ボーダー: 行間 var(--border-light)

5. チップ (StatusChip) 統一

投手/野手/負傷等のステータス表示を統一。StatusChip.vue または v-chip にカスタムクラスを適用。

  • 投手: 藍色背景10%透過 + 藍色テキスト
  • 野手: 萌黄色背景10%透過 + 萌黄テキスト
  • 負傷: 紅色背景8%透過 + 紅色テキスト
  • リコンディショニング: 萌黄色ボーダー + 萌黄テキスト
  • 所属作品: 藍色6%透過 + 藍色ボーダー12%
  • 共通: font-size: 11px, font-weight: 600, border-radius: 4px, padding: 2px 10px

6. 適用対象ページと優先度

  • Phase 1: 共通CSS追加 (App.vue or base.css) + PageHeader/FilterBar改修
  • Phase 2: PlayerCardsView.vue — インラインフィルター→FilterBar移行、カスタムリスト→DataTable移行
  • Phase 3: Players.vue — DataCard内のスタイル統一（既にほぼ準拠、微調整のみ）
  • Phase 4: HomeView/HomePortalView — ダッシュボード系画面の統一
  • Phase 5: 残り全画面（GameRecordListView, ActiveRoster, CostAssignment等）
