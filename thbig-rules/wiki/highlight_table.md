# 好プレー珍プレー表（UP表）
出典: https://thbigbaseball.wiki.fc2.com/wiki/%E5%A5%BD%E3%83%97%E3%83%AC%E3%83%BC%E7%8F%8D%E3%83%97%E3%83%AC%E3%83%BC%E8%A1%A8
取得日: 2026-02-20

## 概要

打撃結果が「UP」の場合に参照する特殊イベント表。2つのd20ダイスで4桁のコードを生成して判定。

## コード生成方法

1. **1つ目のd20** → 千の位と百の位を決定（01〜20 → 0101〜2020の範囲）
2. **2つ目のd20** → 十の位と一の位を決定（01〜20）
3. **4桁コード** = [千百の位][十一の位]
   - 例：1つ目で19、2つ目で07 → コード「1907」

## 球場設定とコード範囲

コード1911〜1914は球場設定によって2分割されている：
- **1911〜1912**: 球場設定 **1112**（センター方向の広さ）
- **1913〜1914**: 球場設定 **1314**（両翼方向の広さ）

### 1901〜1910：特殊プレー

| コード | 結果 |
|--------|------|
| 1901～1910 | 捕手レンジを確認。レンジ0〜1ならホームラン。レンジ2なら打者は1で振り直し。レンジ3以上はレンジチェックレンジチェックでアウトならキャッチャーフライ、それ以外はファールで投球から振り直し |
| 1911〜1912 | 球場確認（ライト方向）。F9aならライトフライ（3塁走者生還、2塁走者3塁へ）。それ以外はホームラン |
| 1913〜1914 | 球場確認（センター方向）。F8aならセンターフライ（3塁走者生還、2塁走者3塁へ）。それ以外はホームラン |
| 1915 | ワイルドピッチ。走者2進塁、打者は振り逃げ（1進塁）。ただし2死以外で1塁に走者がいる時は三振 |
| 1916 | ファーストライナー。すべての走者は飛び出してアウト |
| 1917 | 打球が2塁ベースに直撃しG4f。1塁走者がおらず打者の走力が4以上、または内野前進守備ならIH4 |
| 1918 | G3f。一塁手のレンジが3以下なら2H9 |

## IRC実装メモ

- 打撃結果「UP」判定時に自動的に呼び出し
- 2回のd20ロールで4桁コードを生成
- コード範囲に基づいて処理を分岐
- 1601〜1820：走者の有無を確認してから処理
- 1901〜1920：試合の球場設定（1112/1314）を確認してから対応するコード範囲を参照
- 2018〜2020：追加d20ロールが必要
- エラーチェック（0101〜0920、1609〜1819の一部）：error_check_table.md を参照
- 負傷イベント（2001〜2017など）：怪我チェック処理を呼び出し
