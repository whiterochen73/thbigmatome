# 盗塁

出典: https://thbigbaseball.wiki.fc2.com/wiki/%E7%9B%97%E5%A1%81
取得日: 2026-02-18

## 概要

盗塁は走者が宣言 → 守備側許可 → 2段階判定（スタート判定→成否判定）で実行。ダイス出目と各種補正値で判定。

## 内容

### 盗塁実行の流れ

1. 走者が盗塁を宣言
2. 守備側が許可
3. スタート判定を実施
4. スタート成功時、成否判定を実施

### スタート判定

**構文**: `{背番号}-st{塁}`（例: `10-st2` = 背番号10の選手が2塁への盗塁）

**判定方法**: st値 + ダイス出目に補正を加える

**補正表**

| 状況 | st値補正 | 備考 |
|------|---------|------|
| 基本（二盗） | st値 | そのまま |
| 左投手での二盗 | st値-2 | - |
| 三盗 | st値-4 | - |
| 本盗 | st値-8 | ホームベースへの盗塁 |
| クイック投法使用中 | st値-2 | 投手オプション |
| 出目「2」以下（二盗・三盗） | 補正なし | 出目が低い場合 |
| 出目「1」（本盗） | 補正なし | 本盗時のみ |
| 出目「20」 | 牽制死 | 失敗ではなく牽制で刺される |

**判定ルール**：補正後の値が st値以下なら成功

### 成否判定

**構文**: `{背番号}-ed{塁}`（例: `10-ed2` = 背番号10の選手が2塁での成否判定）

**判定方法**: ed値 + 捕手のT値補正 + ダイス出目を比較

**補正表**

| 項目 | 補正 | 備考 |
|------|------|------|
| 基本（ed値） | ed値 | 走塁値の基礎値 |
| 捕手T値加算 | T値をダイスに加算 | 例：T値+4なら出目に+4 |
| クイック投法使用中 | 捕手T値+2 | 投手オプション時の補正 |
| 本盗（ホーム盗塁） | ed値-8（T値無視） | T値補正は適用されない |
| 出目「20」 | 無条件失敗 | 最高値でも失敗 |

**判定ルール**：ed値以下で成功、超過で失敗

### 盗塁の制約

- **成功後**: 次の塁に進むまで再度盗塁指示不可（ゲームバランス調整）
- **スタート失敗後**: 打撃結果で進塁するまで再盗塁不可

### 盗塁後の行動

**スタートを切った場合**（成功・失敗問わず）:
- 処理後に**両者が作戦変更・選手交代が可能**
- バントを選択できる（**スリーバント扱い**で-5修正。ファール→三振）

**スタートを切れなかった場合**:
- 作戦変更・選手交代は**不可**
- バントは選択可能（同じくスリーバント扱い）
