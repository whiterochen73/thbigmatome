## 1. 先発→先発

### 1-a. 通常

| 中日数 | 疲労P | 負傷チェック |
|--------|-------|------------|
| 中2日以内 | **登板不可** | — |
| 中3日 | 疲労P 0 | **発動** |
| 中4日 | 疲労P −3 | 2回連続で**発動** |
| 中5日 | 疲労P −1 | なし |
| 中6日以上 | カード記載疲労P（通常）| なし |

### 1-b. KO・ノーゲーム時

条件: 4回2/3以下で敗戦 OR 降雨ノーゲーム

| 中日数 | 疲労P | 負傷チェック |
|--------|-------|------------|
| 中3日 | 疲労P −3 | なし |
| 中4日 | カード記載疲労P | 2回連続で**発動** |
| 中5日以上 | カード記載疲労P | なし |

→ 通常より2日短縮（中4日〜がカード値で登板可）

### 1-c. 長イニング敗戦時

条件: カード記載疲労P + 1イニング超過で敗戦した場合

| 中日数 | 疲労P | 負傷チェック |
|--------|-------|------------|
| 中3日 | 疲労P 0 | **発動** |
| 中4日 | 疲労P −4 | 2回連続で**発動** |
| 中5日 | 疲労P −2 | なし |
| 中6日 | 疲労P −1 | なし |
| 中7日以上 | カード記載疲労P | なし |

→ 通常より1日延長（中7日〜がカード値）

## 2. リリーフ→リリーフ（累積イニング管理）

### 2-a. 累積イニング加算・減衰

- **加算**: 登板したイニング数をカウント（出てきたイニング数の合計）
  例: 6回途中〜8回途中 = 累積+3I（6回・7回・8回の3イニング）
  ※ 1イニング内のみで交代した場合は+1I
- **減衰**: 登板しなかった日（休養日・移動日）ごとに
  - 累積3以下の場合: **−2イニング/日**
  - 累積4以上の場合: **−1イニング/日**
  - 下限: 0（マイナスにはならない）

### 2-b. 登板疲労P計算

| 累積イニング数 | 疲労P |
|--------------|-------|
| 0 | カード記載疲労P |
| 1 | 疲労P 0 |
| 2以上 | 疲労状態（※詳細未記載、基本は疲労P 0相当とみなす） |

### 2-c. 中日数による回復

| 中日数 | 疲労P |
|--------|-------|
| 中5日 | 疲労P 0 |
| 中6日以上 | カード記載疲労P |

### 2-d. 負傷チェック発動条件

- 累積3イニング以上での登板時に発動（判定: 1d20で6以下が負傷）

  **判定タイミング**: 試合終了時。判定基準は試合終了時点の累積イニング数。
  発動条件（累積3I以上で登板）と判定基準（試合終了時の累積）は別概念。

  **けが特徴（段階制、試合終了時の累積イニングで判定）**:
  | 試合終了時累積 | けが特徴 |
  |------------|---------|
  | 4I | 5 |
  | 5I | 4 |
  | 6I | 3 |
  | 7I以上 | 2 |

  ※ 累積3Iで登板→最低1I投げる→試合終了時は4I以上。よって試合終了時3Iは発生しない。

### 3-a. 通常

| 中日数 | 疲労P |
|--------|-------|
| 中3日 | 疲労P 0 |
| 中4日以上 | カード記載疲労P |

### 3-b. KO・ノーゲーム後

| 中日数 | 疲労P |
|--------|-------|
| 中2日 | 疲労P 0 |
| 中3日以上 | カード記載疲労P |

### 3-c. 長イニング敗戦後

| 中日数 | 疲労P |
|--------|-------|
| 中4日 | 疲労P 0 |
| 中5日以上 | カード記載疲労P |

### 3-d. スラッシュ選手優遇

スラッシュ（先発/リリーフ兼任）選手: **1日短い間隔でリリーフ回転可能**
- 通常: 中3日→カード値、KO後: 中2日→カード値、長イニング後: 中4日→カード値

## 4. リリーフ→先発

### 4-a. 通常選手

| 中日数 | 疲労P |
|--------|-------|
| 中3日 | 疲労P 0 |
| 中4日 | 疲労P −3 |
| 中5日 | 疲労P −1 |
| 中6日以上 | カード記載疲労P |

### 4-b. スラッシュ選手

直近リリーフ登板の累積イニングに応じて3パターン:

- **累積2I以下**: 中3日→P-1、中4日以上→カード値
- **累積3I**: 中3日→P-3、中4日→P-1、中5日以上→カード値
- **累積4I以上**: 中3日→P0、中4日→P-3、中5日→P-1、中6日以上→カード値

## 5. オープナー起用

**適用条件**: 「オープナー可」アビリティ保有選手のみ。予告先発で「オープナー起用」を明言必須。

### 5-a. 役割分担

| 役割 | 疲労計算ルール |
|------|--------------|
| オープナー本人 | **リリーフ→リリーフ**ルール適用 |
| 第二先発（後続） | **先発→先発**ルール適用 |

### 5-b. オープナー負傷チェック発動条件

- 中2日以内の連続オープナー登板
- 前日登板
- 累積1イニング以上での登板時

→ 判定: 1d20で6以下が負傷（怪我レベル2）

---

## 6. 負傷チェック（共通）

### 発動条件

| # | 条件 |
|---|------|
| 1 | 先発: 疲労P 0での先発 |
| 2 | 先発: 2回連続での中4日先発 |
| 3 | リリーフ: 累積3イニング以上での登板 |

### 判定

- ダイス: 1d20
- 6以下 → **負傷**（けが特徴はリリーフの場合は段階制: §2-d参照。先発・オープナーはLv2）
- 特別ルールでも回避不可

---

## 7. 負傷離脱中の扱い

- **中○日カウントに離脱期間を含めない**（離脱日は中日数に不算入）
- **中10日（離脱除外）** 空ければ疲労なし（カード記載疲労P）で登板可能
- **離脱中は累積イニングの減衰も凍結**: 離脱期間中は累積イニングはそのまま保持（減衰しない）
- **中10日でリセット**: 離脱除外で中10日経過すれば累積イニングも0にリセット（疲労Pカード値復帰と同タイミング）
- 復帰後も負傷前の累積イニングが残った状態で登板する（中10日未満の場合）

## 8. 投手区分の判定ロジック

| 優先順位 | 条件 | 判定 |
|---------|------|------|
| 1 | 直近登板履歴あり | 直近登板の `role` で判定（starter/reliever/opener） |
| 2 | 履歴なし + `is_relief_only = true` | 中継ぎ |
| 3 | 履歴なし + `is_relief_only = false` | 先発 |

**スラッシュ選手の判定**: `starter_stamina IS NOT NULL AND relief_stamina IS NOT NULL AND is_relief_only = false`
