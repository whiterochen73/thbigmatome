# rulebook/data/ — 機械可読形式のテーブル値

AIエージェント・エミュレーターが直接読み込める形式（YAML等）のゲームデータを置く。

## 収録方針

- `../wiki/` のテーブルを機械可読形式に変換したもの
- ルールブック本文（`../01-*.md` 等）から参照する
- 数値テーブルの全セル値はここで管理する
