# バント表
出典: https://thbigbaseball.wiki.fc2.com/wiki/%E3%83%90%E3%83%B3%E3%83%88%E8%A1%A8
取得日: 2026-02-20

## 概要

バント宣言時の成功/失敗を判定するための参照表。先頭走者位置ごとに子ページが分かれている。

- **出目** = d20の目 + バント値
- 先頭走者位置により参照する子ページが異なる（1塁/2塁/3塁）
- バント時は通常のレンジチェック表（range_check_table.md）を使わない。レンジ条件は各子ページに直接記載されている。

## 共通記号一覧

| 記号 | 意味 |
|------|------|
| 成功 | バント成功（送球先が付記される 例: 1-3, 1-5） |
| 成功+ | 2死または無走者時にバントヒット（それ以外は通常成功） |
| 安打 / IH# | バントヒット（#は塁番号: IH1=1塁打相当） |
| 三振・空振り | バント失敗（スリーバント失敗を含む） |
| 併殺 | 走者がアウト（フォース/タッチプレーで状況が異なる） |
| 野選 / FC表 | フィルダースチョイス表（fc_table.md）を参照 |
| R / レンジ | 守備側のレンジに依存 |
| S | 走者走力に依存 |
| RS | 守備レンジと走者走力の両方に依存 |

## 実装メモ

- バント宣言時：バント値 + d20 で出目を計算 → 先頭走者位置の子ページを参照
- **バント時は通常のレンジチェック表（range_check_table.md）を使わない**
- バントマクロ構文: `#-b-1`（1塁先頭）/ `#-b-2`（2塁先頭）/ `#-b-3`（スクイズ）
- スリーバント適用時: 出目に-5補正（さらに加算）
- 内野前進守備時のスクイズ: 出目に-5補正
